# Markdown Live Preview

A modern, responsive Markdown editor with a live side-by-side preview, built entirely with pure HTML, CSS, and JavaScript. Features instant rendering, a handy formatting toolbar, dark/light mode, auto-save, and easy exports in a single portable file.

## Libraries Used
- [**marked.js**](https://github.com/markedjs/marked) – Fast Markdown parser with GFM support
- [**DOMPurify**](https://github.com/cure53/DOMPurify) – Sanitizes HTML to prevent XSS
- [**html2pdf.js**](https://github.com/eKoopmans/html2pdf.js) – Client-side PDF export

All loaded via CDN – no installation required.
