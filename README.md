# Markdown Live Preview

A modern, responsive Markdown editor with a live side-by-side preview, built entirely with pure HTML, CSS, and JavaScript. Features instant rendering, a handy formatting toolbar, dark/light mode, auto-save, and easy exports in a single portable file.

[**Try the Live Demo**](https://lorainecastro.github.io/markdown-live-preview/)

## Key Features
- **Live Side-by-Side Preview** — See beautifully rendered Markdown instantly as you type
- **Modern Design** — Clean cards, rounded corners, subtle shadows, and fully responsive layout
- **Dark/Light Mode Toggle** — Eye-friendly themes with automatic persistence
- **Formatting Toolbar** — Quick buttons for bold, italic, strikethrough, headings, lists, code blocks, links, and quotes
- **Auto-Save** — Your work is saved to the browser automatically
- **Exports** — Download as `.md`, copy rendered HTML to clipboard, or export to PDF
- **Secure Rendering** — Powered by marked.js (GitHub Flavored Markdown) and sanitized with DOMPurify
- **Fully Portable** — Single HTML file, works offline after loading

## Libraries Used
- [**marked.js**](https://github.com/markedjs/marked) – Fast Markdown parser with GFM support
- [**DOMPurify**](https://github.com/cure53/DOMPurify) – Sanitizes HTML to prevent XSS
- [**html2pdf.js**](https://github.com/eKoopmans/html2pdf.js) – Client-side PDF export

All loaded via CDN – no installation required.
